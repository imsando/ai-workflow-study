### 🍀 스터디 목적
1. AI 기반 개발 워크플로우를 이해하고 실습을 통해 익숙해진다.
2. 단순 사용이 아닌, 실제 개발에 적용 가능한 수준까지 활용한다.
3. 각 기능 및 개념을 공유하며 효율적인 사용 방법을 정리한다.

### 🍀 스터디 진행 방식
#### ⚪️ 스터디 일정
- 주 2회 진행 (월 / 목)
- 오전 8시 ~ 9시
#### ⚪️ 스터디 범위
- Session 9부터 진행
- Session 8까지는 개인 학습으로 진행
#### ⚪️ 스터디 방식
- 각자 해당 섹션 내용을 미리 학습
- 핵심 내용 위주로 간단하게 공유 (5분 이내)
- 사용하면서 느낀 점, 문제, 팁 위주로 자유롭게 공유
- 필요 시 간단한 실습 및 화면 공유
#### ⚪️ 디렉토리 구조
```
|-- Session09
  |-- Study
    (작성자).md
  |-- Mission
    (작성자).md
|-- Session10
|-- Session11
...
```

#### ⚪️ 규칙
1.  정리 내용은 마크다운 형식, 내용은 자유
2. 간단한 내용이라도 공유 목적이므로 반드시 업로드
3. 전날 취소 및 당일 불참은 다음 참여에 영향을 줄 수 있음
#### ⚪️ 커밋 규칙
```
[SessionXX]제목(작성자)
```
